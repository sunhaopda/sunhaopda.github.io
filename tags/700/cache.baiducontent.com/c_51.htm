微信朋友圈可以删评论了，但尴尬的是…

确定不再关注此人吗

确定取消

有时候发个朋友圈，总有那么一两条回复让人心里看着别扭，面对不喜欢的评论，只能选择无视或者直接删掉整条朋友圈……

最近，微信悄悄更新了一项新功能――朋友圈中可删除好友评论，该功能可在用户升级至IOS版微信7.0.15后体验，不过，安卓用户暂不支持体验。

操作上很简单，在自己发布的朋友圈下，只要长按某一条留言，就可以删除这条评论的内容。

为了防止手误，微信还设置了二次提醒：是否确定要删除该评论？

留言删除之后，在自己的朋友圈下将不再有这条留言。不过尴尬的是，如果评论你的人，也用最新版微信，再看回你这条朋友圈，会看到“评论已被删除”的提醒，对方可以通过长按来删除这条提醒。
